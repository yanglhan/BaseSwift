# AYSeg
简单好用的分页UI框架，支持：

* 页面分页
* 分类、标签页

## Installation and Setup

*最新版本支持Swift 5*

AYSeg works on Xcode 7-11 and above, iOS 9-13 and above.

## Requirements

  * IOS 9.0+
  * Xcode 10+
  * swift 4+

### CocoaPods

Podfile文件配置：

Swift 4.0  --> pod 'AYSeg', '0.0.9'

Swift 5.0 --> pod 'AYSeg', '0.5.2'
